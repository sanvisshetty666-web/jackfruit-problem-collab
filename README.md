## Project Overview
This is a modern GUI application developed in Python using the Tkinter framework. It provides a simple, intuitive interface for performing classical Caesar Cipher encryption and decryption.

### Key Details
* **Project Title:** Caesar Cipher Encryption & Decryption Tool Using Tkinter
* **Language:** Python 3.x
* **GUI Framework:** Tkinter

## ‚ú® Features
1.  **Encryption & Decryption:** Supports both modes with any shift key (1-25).
2.  **Flexible Input:** Allows direct text input or loading from a external `.txt` file.
3.  **Theming:** Includes a one-click toggle for Light Mode and Dark Mode.
4.  **Output Utilities:** Buttons to **Copy** the output to the clipboard or **Download** the result as a `.txt` file.
